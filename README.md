# lockbox
# EE 1301 Final Project
# Team Members: Beau Lindley, Sebastian Valine

Microcontrolled lockbox; takes sequence of inputs from pushbutton input and compares to stored key. If correct, it unlocks the box by turning a servo inside the box. Box will relock when button held.

Primary code for the the Photon can be found in the firmware/src/ folder.

Current issues/tasking, as well as issue history, can be found on the issues tab.
